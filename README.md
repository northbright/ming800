ming800是一个[Golang](https://golang.org)包，提供适用于[旧单机版本](/#适用版本)明日管理系统的API接口。

#### 适用版本
* 2012年单机安装版本
* 只有1个校区（总部）

#### 功能
* 按电话号码，姓名搜索学生信息
* 获取当前所有专业与开设班级
* 获取一个班级所有学生信息
* 获得当前在读所有学生信息
